# Batch Video Editor

A powerful desktop application for batch video processing using FFmpeg and PyQt5.

## Features
- Batch processing with queue management
- GPU acceleration (NVENC) support
- Image, video, and text overlays with expression-based scaling
- Intro/Outro video concatenation
- Flexible codec and quality settings
- New trimming functionality: cut seconds from start/end

## Requirements
- Python 3.8+
- PyQt5
- ffmpeg-python
- FFmpeg installed and added to system PATH

## Installation
```bash
pip install -r requirements.txt
```

## Usage
Run the application:
```bash
python main.py
```
